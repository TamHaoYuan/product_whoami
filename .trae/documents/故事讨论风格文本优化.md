# 虚像贴吧 - 文本风格优化计划

## 概述
将网站的文本改为更像故事中的人物讨论故事的氛围，参考《恶意》中网友讨论《恶意》中的《萤火虫》的风格。

## 目标
- 将"帖子"改为更具故事讨论感的词汇（如"话题"、"分享"、"议论"）
- 修改表单文案，使其更像故事人物的讨论语气
- 更新导航和侧边栏文案
- 保持功能不变，仅修改文案

## 实施步骤

### 1. 修改配置文件文案
- 修改 `src/config/site.json`
- 将"发帖"改为"分享故事"或"议论"
- 将"帖子列表"改为"最近的议论"

### 2. 修改发帖表单文案
- 修改 `src/components/islands/PostForm.tsx`
- "帖子标题" → "分享标题"或"议论题目"
- "帖子内容" → "详细内容"或"你的看法"
- "发布帖子" → "分享出去"

### 3. 修改导航栏文案
- 修改 `src/components/Navigation.astro`
- "发帖" → "分享故事"
- "帖子列表" → "最近的议论"

### 4. 修改侧边栏文案
- 修改 `src/components/Sidebar.astro`
- "搜索帖子" → "搜索话题"

### 5. 修改帖子列表和详情页文案
- 修改 `src/components/islands/PostList.tsx`
- 修改 `src/pages/post.astro`
- "发表帖子" → "分享故事"
- "帖子不存在" → "这道议论似乎消失了"

### 6. 修改首页文案
- 修改 `src/pages/index.astro`
- "最新帖子" → "最近的议论"或"新鲜的话题"

### 7. 修改发帖页面文案
- 修改 `src/pages/create.astro`
- "发表新帖子" → "分享你的故事"

## 验收标准

### AC-1: 文案风格统一
- **Given**: 用户浏览网站
- **When**: 查看各个页面时
- **Then**: 所有文案都呈现故事人物讨论的氛围
- **验证**：`human-judgment`

### AC-2: 功能保持不变
- **Given**: 用户使用发帖功能
- **When**: 填写表单并发布时
- **Then**: 功能正常，数据正常保存
- **验证**：`programmatic`

## 关键词替换对照表
| 原词汇 | 新词汇 |
|--------|--------|
| 帖子 | 议论 / 话题 / 分享 |
| 发帖 | 分享故事 / 展开议论 |
| 帖子列表 | 最近的议论 |
| 帖子标题 | 分享标题 |
| 帖子内容 | 详细内容 |
| 发布帖子 | 分享出去 |
| 搜索帖子 | 搜索话题 |
| 帖子不存在 | 这道议论似乎消失了 |
