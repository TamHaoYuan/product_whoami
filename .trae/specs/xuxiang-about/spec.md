# 虚像茶馆 - 简介页面规格

## Why

虚像茶馆需要一个专门的介绍页面，让访问者了解这个以非上帝视角讨论故事的独特空间是什么，以及它的理念和用途。

## What Changes

* 创建 `/about` 页面作为虚像茶馆的简介页面

* 页面采用故事中人物对话的氛围设计

* 包含茶馆的起源、理念和使用说明

* 保持与整体网站风格一致

## Impact

* Affected specs: 虚像茶馆现有功能

* Affected code: 需要创建 `src/pages/about.astro`

## ADDED Requirements

### Requirement: 简介页面

虚像茶馆应提供专门的简介页面，让访问者了解茶馆的用途和理念。

#### Scenario: 访问简介页面

* **WHEN** 用户访问 /about 页面

* **THEN** 显示虚像茶馆的简介内容，包括：

  * 茶馆的定位和理念

  * "非上帝视角讨论故事"的含义

  * 简单的使用指南

## MODIFIED Requirements

### Requirement: 导航栏更新

导航栏应包含"关于茶馆"链接。

## REMOVED Requirements

无

## 页面内容设计

### 1. 页面标题

* 虚像茶馆简介

### 2. 主要内容

```
【关于虚像茶馆】

欢迎来到虚像茶馆。

这里是一个特别的场所——故事中的人物们聚在一起，议论故事的地方。

不同于读者从外部观察故事，这里是角色们从内部讨论他们所处的世界。
或者是任何一个故事中的角色，对剧情发表自己的看法……

在这里，你可以是任何故事中的任何人物，
以角色的身份与其他故事的角色一起，
从非上帝视角讨论那些令人难忘的故事、角色和剧情。

【使用说明】
- 无需注册，选择你的身份即可开始议论
- 可以自定义用户名和头像
- 管理员和站长需要密码验证
- 尽情分享你对故事的见解吧
```
